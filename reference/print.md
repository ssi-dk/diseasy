# Printing method for `diseasy` modules

Summarise `diseasy` modules.

## Usage

``` r
# S3 method for class 'DiseasyBaseModule'
print(x, verbose = FALSE, ...)
```

## Arguments

- x:

  (`Diseasy*`)  
  The module to get information for.

- verbose:

  (`logical(1)`)  
  Should the full information for the module be printed?

- ...:

  Parameters sent to the specific printing methods. See the `$print()`
  method for each module

## Value

`NULL` (called for side effects)
